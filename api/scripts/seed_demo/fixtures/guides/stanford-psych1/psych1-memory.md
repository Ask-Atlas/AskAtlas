# Memory

Memory is not a single faculty. It's a collection of partially
dissociable systems with different time scales, neural substrates, and
failure modes. The phrase "I forgot" smuggles together at least four
distinct possibilities: it never encoded, it was encoded but
consolidated weakly, it consolidated but the retrieval cue is wrong,
or it was retrieved and reconstructed inaccurately. Telling them apart
is most of what this unit is for.

## A working model

A useful (and famous) framework is the **Atkinson-Shiffrin
multi-store model**: information passes from sensory memory to
short-term memory to long-term memory, with attention and rehearsal
gating each transition.

- **Sensory memory** — very brief (≈ 250 ms for iconic visual memory,
  a few seconds for echoic auditory memory). Holds the raw sensory
  trace of everything you just sensed, before attention selects.
- **Short-term memory (STM)** — capacity famously about 7 ± 2 items
  (Miller); duration without rehearsal ≈ 15-30 seconds.
- **Long-term memory (LTM)** — effectively unbounded capacity, can
  last a lifetime.

Modern accounts replace simple STM with **working memory**
(Baddeley) — an active workspace with subsystems for verbal material
(phonological loop), visual/spatial material (visuospatial sketchpad),
and a central executive that allocates attention. Working memory is
*processing* memory, not just storage; it's what you use to do mental
arithmetic.

## Long-term memory has structure

LTM is itself layered. The first split is conscious vs. unconscious.

```
Long-term memory
├── Explicit (declarative) — conscious, verbalisable
│   ├── Episodic — personal events, time-and-place tagged
│   └── Semantic — facts about the world, concepts, vocabulary
└── Implicit (non-declarative) — unconscious
    ├── Procedural — skills (riding a bike)
    ├── Priming — recent exposure speeds processing
    └── Classical conditioning — associations
```

The strongest evidence for these subsystems is **double dissociation**
in patients. H.M., after bilateral hippocampal removal in 1953, could
not form new explicit memories — but he could learn new motor skills
normally without remembering ever having practised them. Patients with
striatal damage often show the opposite pattern. Two systems, two
substrates.

## Encoding

Encoding is the conversion of an experience into a memory trace. Three
findings matter most:

- **Levels of processing** (Craik & Lockhart) — *deeper* processing
  (semantic, meaningful) yields better recall than *shallow*
  processing (rhyme, surface features). "Does this word fit in this
  sentence?" produces better memory than "is this word in capital
  letters?"
- **Spacing effect** — material studied with breaks between sessions
  is remembered better than the same total time crammed together.
  This is one of the most robust findings in cognitive psychology and
  the load-bearing argument against last-night-only studying.
- **Generation effect** — actively producing the answer (or even
  attempting to and failing) beats passively re-reading it. This is
  why retrieval practice (active recall) outperforms re-reading on
  almost every memory measure researchers have tried.

> Two study sessions of 30 minutes a week apart will outperform one
> 60-minute session almost every time. Same time, different memory.

## Storage and consolidation

Once encoded, memories don't sit static. **Consolidation** is the
multi-hour-to-multi-year process by which a labile trace becomes a
stable one. Several mechanisms operate at different time scales:

- **Synaptic consolidation** (minutes to hours) — long-term
  potentiation strengthens synapses through gene expression and
  protein synthesis.
- **Systems consolidation** (months to years) — hippocampus-dependent
  memories gradually become independently retrievable from cortex.
  Older memories survive hippocampal damage that wipes out recent
  ones.

Sleep is implicated heavily in consolidation. Slow-wave sleep appears
to favour declarative memory; REM appears to favour procedural and
emotional memory. Sleep-deprive someone after learning and memory for
that material suffers. Pulling an all-nighter to study trades the
encoding gain for the consolidation loss; the trade is usually not
worth it.

## Retrieval

Retrieval is not playback. It is the active reconstruction of a
memory at the time you ask for it, using whatever cues are available,
plus prior knowledge and current context.

- **Encoding specificity principle** — retrieval is best when the cues
  available at recall match those present at encoding. If you study in
  one room, you tend to recall slightly better in the same room.
- **State-dependent and mood-congruent memory** — recall is better
  when internal state at retrieval matches encoding (caffeinated to
  caffeinated, sad to sad).
- **Recognition vs. recall** — recognition (multiple choice) is
  almost always easier than free recall, because the test itself
  supplies retrieval cues.
- **Tip-of-the-tongue** — partial retrieval (you know the first letter
  and number of syllables but not the word) is direct evidence that
  retrieval is not all-or-none.

## Forgetting

The classic **Ebbinghaus forgetting curve** is approximately
exponential: most forgetting happens fast in the first hour or two,
then the rate slows. Several mechanisms contribute:

- **Decay** — memory traces weaken with time if not retrieved.
  Probably real but contested as the dominant cause.
- **Interference** — other memories disrupt access.
  - **Proactive** — old learning interferes with new (your old phone
    number keeps surfacing when you try to recall the new one).
  - **Retroactive** — new learning interferes with old (you can't
    remember last year's locker combination after learning this
    year's).
- **Retrieval failure** — the trace is intact but the right cue isn't
  available. Recall later under different cues and the memory comes
  back.
- **Motivated forgetting / repression** — Freud's contribution.
  Empirically slippery; some inhibition effects are well-documented,
  but full Freudian repression remains controversial.

## Memory is reconstructive

The single most important finding about everyday memory is that it is
**reconstructive**. Each retrieval re-encodes the memory, and during
that re-encoding the memory can be subtly altered.

Loftus's **misinformation effect** experiments are the canonical
demonstration. Participants watched a video of a car accident, then
were asked questions that contained subtle misinformation ("How fast
were the cars going when they *smashed* into each other?" vs. *hit*).
The wording shifted later memory of the same video — speed estimates
went up, and a non-trivial fraction of participants later "remembered"
broken glass that was never present.

Practical consequences are large:

- **Eyewitness testimony** is far less reliable than juries
  intuitively assume, especially across racial lines and after leading
  questioning.
- **False memories** for plausible childhood events (a balloon ride, a
  trip to a hospital) can be implanted in laboratory studies in a
  noticeable minority of participants.
- **Flashbulb memories** — vivid memories of emotionally charged
  public events (where you were when X happened) feel certain but are
  no more accurate than ordinary memories on careful follow-up.

This is why the {{GUIDE:psych1-research-methods}} discipline matters
so much in clinical and legal contexts: confidence is not calibration.

## Practical study takeaways

If your goal is durable PSYCH 1 knowledge:

1. **Space your study sessions.** Multiple shorter sessions beat one
   marathon.
2. **Test yourself, don't re-read.** Cover the material and try to
   produce it. Use the quizzes in this course.
3. **Process meaningfully.** Connect new material to things you
   already know.
4. **Sleep.** The memory you encoded today will be more available
   tomorrow if you sleep tonight.
