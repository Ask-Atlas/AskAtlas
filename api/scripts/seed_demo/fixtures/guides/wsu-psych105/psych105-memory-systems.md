# Memory Systems

If perception is how the world gets in, memory is how it stays. The
PSYCH 105 unit on memory leans on a single dominant model — the
Atkinson–Shiffrin multi-store framework — supplemented by Baddeley's
working-memory account and Tulving's episodic/semantic split. Get the
vocabulary clean and you can answer almost every memory question on
the midterm by tracing where in the system the failure occurred.

## The three-store model

> **Atkinson–Shiffrin (1968)** — information flows through three
> stores: a brief sensory register, a limited-capacity short-term
> store, and a vast long-term store. Attention moves items from
> sensory to short-term; rehearsal moves them from short-term to
> long-term.

| Store | Capacity | Duration | Encoding |
|---|---|---|---|
| Sensory | Very large (entire perceptual field) | ~250 ms (visual), ~3 s (auditory) | Iconic / echoic |
| Short-term | 7 ± 2 chunks (Miller) | ~15–30 s without rehearsal | Mostly acoustic |
| Long-term | Effectively unlimited | Lifetime | Mostly semantic |

The capacity drop is dramatic. Sperling's partial-report procedure
showed that the sensory register holds essentially everything for a
quarter-second, but the short-term bottleneck only lets a handful of
items through.

### Working memory (Baddeley)

Baddeley reframed "short-term memory" as an active workspace with
distinct components:

- **Phonological loop** — acoustic/verbal rehearsal.
- **Visuospatial sketchpad** — visual and spatial information.
- **Central executive** — attention controller; allocates resources.
- **Episodic buffer** (added 2000) — integrates information across
  modalities and links to long-term memory.

This model is the one cognitive psychologists actually use today; the
classic three-store diagram is taught for its historical importance.

## Long-term memory subdivisions

Long-term memory is not a single bucket. Tulving's split:

- **Explicit (declarative)** — accessed with conscious recall.
  - **Semantic** — facts ("Olympia is the capital of Washington").
  - **Episodic** — personally experienced events ("Where I was when
    the lights went out last March").
- **Implicit (non-declarative)** — accessed without conscious recall.
  - **Procedural** — skills and habits (riding a bike).
  - **Priming** — recent exposure facilitates later processing.
  - **Classical conditioning** — see
    {{GUIDE:psych105-classical-and-operant-conditioning}} for the
    learning side of this.

Patients with hippocampal damage (HM is the canonical case) lose the
ability to form new explicit memories but can still acquire new
procedural skills. That double dissociation is the strongest
neurological evidence for the explicit/implicit split.

## Encoding — getting it in

Three encoding moves consistently improve later recall:

1. **Elaborative rehearsal** — connect new material to what you
   already know. Beats simple repetition (maintenance rehearsal).
2. **Levels of processing** (Craik & Lockhart) — deeper, semantic
   processing produces stronger traces than shallow, perceptual
   processing.
3. **Self-reference effect** — material processed in relation to
   yourself is remembered better than material processed in
   relation to others.

Distributed practice ("spacing") beats massed practice ("cramming")
across nearly every memory experiment ever run. The
{{FILE:wsu-psych105-memory-systems-cheatsheet}} cheatsheet has the
spacing-effect curves you'll see in lecture.

## Storage — keeping it

Long-term storage is reorganized over time. The hippocampus is
critical for forming new explicit memories, but consolidated
memories migrate to cortical areas — which is why HM could recall
his pre-surgery childhood but not his post-surgery breakfasts.

Sleep, particularly REM and slow-wave sleep, plays a documented
role in consolidation. Pulling an all-nighter is precisely the
wrong thing to do the night before an exam.

## Retrieval — getting it back

Retrieval is reconstructive, not playback. Each recall partially
re-encodes the memory, which is why memories can drift toward
later suggestions.

Retrieval cues matter:

- **Encoding specificity** — recall is best when retrieval
  conditions match encoding conditions. Studied at a desk, tested
  at a desk → better recall than studied at a desk, tested in a
  pool.
- **State-dependent memory** — physiological state at encoding
  matches state at retrieval (the "drunk what you learned drunk"
  effect, with weaker but real evidence).
- **Mood-congruent memory** — current mood biases which memories
  come to mind.
- **Tip-of-the-tongue** — partial activation; you know that you
  know it.

## Forgetting

Several distinct mechanisms:

- **Decay** — the trace weakens with time (mostly relevant for
  short-term memory).
- **Interference** — other memories compete.
  - **Proactive** — old learning interferes with new (last
    semester's locker combination intrudes on this semester's).
  - **Retroactive** — new learning interferes with old (you can
    no longer recall the old combination).
- **Retrieval failure** — the trace is intact but inaccessible
  without the right cue.
- **Motivated forgetting** — repression as a contested
  mechanism.

## The seven sins of memory (Schacter)

A useful taxonomy of what *goes wrong*:

1. **Transience** — fading over time.
2. **Absent-mindedness** — encoding failure due to inattention.
3. **Blocking** — temporary inaccessibility (TOT).
4. **Misattribution** — right info, wrong source.
5. **Suggestibility** — incorporation of misinformation.
6. **Bias** — current beliefs distort recall of the past.
7. **Persistence** — unwanted intrusive memories (PTSD).

## The misinformation effect

Loftus's classic demonstration: show participants a film of a car
accident, then ask either "How fast were the cars going when they
**hit**?" or "...when they **smashed**?" The smashed group reports
higher speeds and is more likely to falsely remember broken glass
in the original film. Eyewitness memory is malleable, and leading
questions can plant entirely fabricated details. This is why modern
police interview protocols use **cognitive interviews** with open
questions and multiple retrieval attempts.

## Strategies that actually work

Take any opportunity in PSYCH 105 to practice these:

- **Spaced retrieval practice** (test yourself across days)
- **Elaborative interrogation** ("why is this true?")
- **Self-explanation** of worked examples
- **Mnemonics** for arbitrary lists (method of loci, acronyms)
- **Sleep** between study and test

For the exam itself, walk in well-rested. The data is overwhelming.

## Study cues

- If a question asks about a 7±2 capacity, the answer is short-term.
- If a question describes a brief, sensory-rich snapshot, it's
  iconic memory.
- If a question describes a skill that survives amnesia, it's
  procedural (implicit).
- If a question shows a leading question changing recall, that's
  suggestibility / the misinformation effect.
