# Memory Systems Cheatsheet

Human memory is best understood through the **Atkinson-Shiffrin multi-store model** (1968), which proposes three distinct stages: sensory memory, short-term memory (STM), and long-term memory (LTM). Baddeley & Hitch (1974) later refined STM into the **working memory model**, adding the phonological loop, visuospatial sketchpad, central executive, and episodic buffer.

## Comparing the Three Stores

| Store | Capacity | Duration | Encoding | Example |
|---|---|---|---|---|
| Sensory memory | Very large (all sensory input) | 0.25–4 seconds | Raw sensory (iconic, echoic) | The "trail" of a sparkler |
| Short-term / Working | 7 +/- 2 items (Miller, 1956); ~4 chunks (Cowan, 2001) | 15–30 seconds without rehearsal | Primarily acoustic | Holding a phone number |
| Long-term memory | Essentially unlimited | Minutes to a lifetime | Primarily semantic | Remembering your address |

## Long-Term Memory Subtypes

LTM splits into **explicit (declarative)** and **implicit (non-declarative)** systems (Squire, 1992):

- **Explicit** — conscious recall
  - *Episodic*: personal events ("my high school graduation")
  - *Semantic*: facts and concepts ("Paris is the capital of France")
- **Implicit** — unconscious recall
  - *Procedural*: motor skills ("riding a bike")
  - *Priming*: prior exposure influencing response
  - *Classical conditioning*: learned associations

## Encoding Strategies

Craik & Lockhart's (1972) **Levels of Processing** framework argues that deeper (semantic) processing produces better retention than shallow (visual or acoustic) processing.

- **Elaborative rehearsal**: linking new info to existing schemas (more durable than rote rehearsal)
- **Chunking**: grouping items into meaningful units (e.g., phone number as 3-3-4)
- **Mnemonic devices**: method of loci, acronyms, the peg-word system
- **Dual coding** (Paivio): combining verbal and visual representations
- **Self-reference effect**: information related to the self is better remembered

## Retrieval

Retrieval is reconstructive, not reproductive (Bartlett, 1932). Key concepts:

- **Encoding specificity principle** (Tulving & Thomson, 1973): retrieval is best when conditions match encoding
- **Context-dependent memory**: same physical environment aids recall
- **State-dependent memory**: same internal state (mood, substances) aids recall
- **Recognition vs. recall**: recognition (multiple choice) is easier than free recall
- **Tip-of-the-tongue phenomenon**: retrieval failure, not storage failure

## Forgetting

Ebbinghaus's **forgetting curve** shows rapid decay that levels off over time. Causes include:

1. **Decay** — unused traces fade
2. **Interference** — proactive (old blocks new) or retroactive (new blocks old)
3. **Retrieval failure** — information stored but not accessed
4. **Motivated forgetting** — repression (Freud) or suppression

## Study Application

Use **spaced practice** (distributed over days) and **retrieval practice** (self-testing) rather than massed rereading. Roediger & Karpicke (2006) demonstrated the **testing effect**: students who self-tested outperformed those who restudied on delayed recall.
