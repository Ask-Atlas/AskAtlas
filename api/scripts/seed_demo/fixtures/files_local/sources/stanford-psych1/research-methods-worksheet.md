## Part 2 — Experimental vs. Correlational

Mark each scenario as **Experimental (E)** or **Correlational (C)** and give a one-sentence justification.

| # | Scenario | E or C? | Why? |
|---|---|---|---|
| a | Festinger & Carlsmith (1959) paid participants $1 or $20 to lie, then measured attitude change. | | |
| b | A researcher compares GPA and hours of sleep reported by 500 undergraduates. | | |
| c | Bandura (1961) randomly assigned children to watch an adult act aggressively or calmly toward a Bobo doll. | | |
| d | A journalist reports that countries with more ice cream sales have more drownings. | | |

Compare the two designs:

| Feature | Experimental | Correlational |
|---|---|---|
| Manipulation of IV | Yes | No |
| Random assignment | Required for causal claims | Not possible |
| Supports causal inference | Yes | No — third variables possible |
| Ethical/practical reach | Narrower | Broader |
| Typical statistic | t, F | r, β |
